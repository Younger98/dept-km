# 快速開始

歡迎開始使用本知識庫！

## 什麼是知識庫？

知識庫是一個集中存放技術文件、API 說明、操作手冊的中心，讓團隊成員可以快速找到所需的資訊。

## 如何使用

1. **瀏覽目錄** - 使用左側導航欄
2. **搜尋** - 使用搜尋功能查找特定內容
3. **部門專區** - 每個部門有獨立頁面

## 文件規範

請參考 [風格指南](../templates/style-guide.md)

## 常見問題

**Q: 如何新增文件？**
A: 請參考文件撰寫流程，建立 PR 後由 Reviewer 審核。

**Q: 如何回報錯誤？**
A: 請在 GitHub Repository 開 Issue。
